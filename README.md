# catch-the-dice-reflexes-game
A reflexes game featuring a dice that changes faces.

# Requirements
• Windows 7 SP1 or newer

• .NET Framework 4.7.2 or newer

# Download
Click the Code button, then click on "Download ZIP"

# Execution
Run DiceReflexGame.exe, located in DiceReflexesGame\DiceReflexGame\bin\Debug

# Usage
• Enter your name in the field located on the top side of the app.

• Start the game by clicking on Game → New Game. Alternatively, hit Ctrl + N.

• The dice will start moving across the screen. Every time it changes position, its face will change. Click on the dice to score points. The points scored are as many as the dice face indicates. Please note: once you score points by clicking on the dice, you won’t be able to score again until it changes position.

• You have 60 seconds to score as many points as you can!

# Other Features

• 3 difficulty levels: Easy, Medium and Hard. Medium is selected by default.

• View previous attempts: see who played before you, their score, when they played (date & time), and the difficulty level they played at.

• Pause button: pause the game anytime you want during gameplay.

• High score: view your personal best, and try to surpass it!
